Alexnet Hw&Sw Optimization            
===============================

1.설명                 
------
   
* FPGA연구실에서 만든 하드웨어 가속기를 이용하여 CPU와 병렬로 Alexnet을 연산시킴으로서 최적화시키는 프로젝트           
            
   
2.사용 기술     
------        
* 가속기를 CPU에 연결하기 위한 하드웨어 디자인     
 - DMA연결    
 - Interrupt선 연결    
* ISR(Interrupt Service Routine)작성        
* 동시에 계산하기 위한 Pipeline 설계                    

3.성적       
------    
* 11개의 팀 중 1등
